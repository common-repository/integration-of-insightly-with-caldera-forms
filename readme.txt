Insightly and Caldera Forms Integration

== Description ==

The Integration of Insightly and Caldera Forms plugin lets you add a new Insightly Processor to Caldera form. It automatically syncs data from your Caldera form to your Insightly CRM when the form is submitted.

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "Integration of Insightly and Caldera Forms".
3. Click install.
4. Click activate.

== Screenshots ==
1. Create First Name, Last Name and Email fields in the Caldera Forms plugin and then click on Save Form
2. After creating the form, Goto Processors and Choose the Insightly Integration Processor
3. Enter the Insightly API key and API URL and choose the correct variables for the first name, last name and email fields and then click on Save Form button
4. Use the Preview Button to Preview the form and fill the form
5. After submitting the form, You can check the data in the Insightly's Contacts page
